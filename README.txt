/***********************************/
1. Follow the steps as shown below:
/***********************************/

        1. Compile WordCount.c using the below command:
                
              gcc -pthread -o WordCount WordCount.c
              
        2. Execute the Program using the below command:
        
              ./WordCount
              
        3. Choose the options as directed:
                1 = Fast word count
                2 = Slow word count


/***********************************************/              
Limitations or issues while solving the problem:
/***********************************************/

        1. C does not allow for inheritance and hence cannot use extends in the solution 
        
        2. C language does not have exceptions
